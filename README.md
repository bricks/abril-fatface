## Abril Fatface

[Abril Fatface](http://hellohappy.org/beautiful-web-type/#headline)

## Install

```bash
$ npm install abril-fatface
```

## Preview

![](https://cldup.com/Cpg2UkS3tG.png)

## Usage

Require and mix with any Brick:

```js
var Brick = require('brick')
var AbrilFatface = require('abril-fatface')

module.exports = Brick(AbrilFatface)
```
